## Mission Types

### ❓ Quiz Scenario
- 3 questions per mission
- Each question: story setup → scenario description → 3 options (A/B/C)
- Immediate feedback per answer with explanation
- Pro tip revealed after answering
- Points awarded only for correct answers

### 🔍 Spot the Threat
- Interactive rendered email / SMS / website mockup
- Click suspicious elements to identify threats
- Real-time feedback popover explains WHY each element is a threat
- Score accumulates per threat found
- "Submit" reveals any missed threats (highlighted in red)
- Point deduction system not used — encourages exploration

### 🌿 Decision Tree
- Branching narrative with 2–3 choices per node
- Choices lead to different outcomes (hero / partial / bad ending)
- Points awarded per good decision
- Final ending card explains the real-world lesson
- Multiple paths possible — replay value built in

## Singapore Context

All adult and senior scenarios reference Singapore-specific services:
- **PayNow / PayLah!** — payment platform referenced in scam scenarios
- **SingPass** — national identity platform
- **SPF** — Singapore Police Force (official: 1800-255-0000)
- **ScamShield** — anti-scam app & hotline (1800-722-6688)
- **ScamAlert.sg** — official scam advisory portal
- **DBS / OCBC / UOB** — major Singapore banks
- **NTUC FairPrice** — prize scam reference
- **CPF / IRAS / MAS** — government agencies used in impersonation scams
